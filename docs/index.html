Desc¶

一个超简单的命令行加密货币钱包,支持btc和eth,硬分叉完后视情况添加bch的支持

keywords: wallet

Feature¶

  • 完全无状态,不保存任何用户信息,用户自己保存私钥地址这类信息
  • 支持查看balance,新建一个私钥地址对以及指定账户进行转账

Example¶

ecw -h

Install¶

  • python -m pip install EC_CMD_Wallet
